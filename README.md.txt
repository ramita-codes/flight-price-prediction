# Flight Price Prediction ✈️

This project predicts airline ticket prices using machine learning.

## Dataset
The dataset contains information about airline flights including:

- airline
- source city
- destination city
- departure time
- arrival time
- stops
- class
- price

# Flight Price Prediction
Dataset format: CSV

## Machine Learning Model
Random Forest Regressor

## Steps in the Project

1. Data preprocessing
2. Encoding categorical variables
3. Train-test split
4. Model training
5. Prediction
6. Visualization of predicted vs actual prices

## Libraries Used

- pandas
- numpy
- matplotlib
- scikit-learn

## Output

The model predicts flight prices and visualizes:

Actual Price vs Predicted Price.
